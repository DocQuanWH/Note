# 基数排序
假如有如下待排序数列：
> 73, 22, 93, 43, 55, 14, 28, 65, 39, 81

根据个位的数值，可以将其放入0 ~ 9的桶中：

| 0 | | | | |
| --- | --- | --- | --- | --- |
| **1** | 81 |  |  |  |
| **2** | 22 |  |  |  |
| **3** | 73 | 93 | 43 |  |
| **4** | 14 |  |  |  |
| **5** | 55 |  |  |  |
| **6** | 65 |  |  |  |
| **7** |  |  |  |  |
| **8** | 28 |  |  |  |
| **9** | 39 |  |  |  |
