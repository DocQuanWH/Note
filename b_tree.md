# B树
**B树（B-Tree）**是为磁盘等辅助存取设备设计的一种**平衡查找树**，它实现了以**O(logn)时间复杂度**执行查找、顺序读取、插入和删除操作。由于B树和B树的变种在**降低磁盘I/O操作次数**方面表现优异，所以经常用于设计文件系统和数据库。
